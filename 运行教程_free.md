# 录屏工具运行教程（免费版）

## 一、环境准备

### 1. 安装 Python
- 确保你的系统已安装 Python 3.7 或更高版本
- 下载地址：https://www.python.org/downloads/
- 安装时请勾选 "Add Python to PATH"

### 2. 安装依赖包

打开命令行工具（Windows 下为 cmd 或 PowerShell），进入项目目录，执行以下命令：

```bash
cd d:\代码存档\zbtip\zbtipfree
pip install -r requirements_free.txt
```

如果没有 requirements_free.txt 文件，请执行以下命令手动安装依赖：

```bash
pip install opencv-python pyautogui numpy Pillow pyaudio keyboard
```

## 二、运行程序

### 直接运行

在项目目录下执行：

```bash
cd d:\代码存档\zbtip\zbtipfree
python screen_recorder_free.py
```

### 打包运行

如果已打包为可执行文件，直接双击 `直播录屏标记助手.exe` 即可运行。

## 三、功能使用

### 1. 基本录制

- **开始录屏**：点击主界面的 "开始录屏" 按钮
- **暂停录屏**：点击 "暂停录屏" 按钮（录制中可用）
- **继续录屏**：再次点击 "继续录屏" 按钮
- **结束录屏**：点击 "结束录屏" 按钮完成录制

### 2. 标记进度

- **添加标记**：点击 "标记进度" 按钮或按空格键，在当前位置添加黄色标记
- **标记显示**：标记会显示在进度条上，鼠标悬停可查看标记信息
- **点击跳转**：点击进度条上的标记可快速跳转到对应位置
- **无限使用**：免费版标记次数无限制

### 3. 视频截取

- **开始截取**：点击 "截取视频" 按钮
- **完成截取**：点击 "完成截取" 按钮，截取从开始到当前的视频片段
- **取消截取**：再次点击 "截取视频" 按钮可取消截取模式
- **无限使用**：免费版截取功能无限制

### 4. 录制记录

- **查看记录**：点击 "录制记录" 按钮，查看所有历史录制
- **打开目录**：选中记录后点击 "打开选中目录"，可打开该录制的文件夹

### 5. 视频资料库

- **打开资料库**：点击 "视频资料库" 按钮
- **浏览视频**：查看所有已保存的视频文件
- **快速访问**：双击文件夹可快速定位视频位置

## 四、文件保存位置

录制的视频文件保存在应用程序所在目录的 `recordings` 文件夹中：

```
zbtipfree/
├── recordings/
│   └── 20260503_143000/          # 录制会话目录（以时间命名）
│       ├── recording_20260503_143000.avi   # 原始录制视频
│       ├── markers.json           # 标记数据
│       └── 截取视频/             # 截取的片段目录
│           ├── clip_01.avi       # 截取片段1
│           ├── clip_02.avi       # 截取片段2
│           └── clip_info.json    # 片段信息
```

## 六、常见问题

### 1. 录制视频没有声音
- 本版本暂不支持音频录制（后续版本会添加）

### 2. 录制后找不到视频文件
- 视频文件保存在应用程序所在目录的 `recordings` 文件夹中
- 每个录制会话会创建一个单独的文件夹，命名格式为 "YYYYMMDD_HHMMSS"

### 3. 标记不显示在时间轴上
- 确保已正确加载录制视频
- 标记信息保存在对应会话目录的 `markers.json` 文件中

### 4. 视频截取失败
- 确保录屏过程中没有暂停太久
- 确保磁盘有足够空间

## 七、注意事项

- 录制过程中尽量避免操作过多的应用程序，以保证录制质量
- 长时间录制可能会占用大量磁盘空间，请确保有足够的存储空间
- 如需更改保存路径，请修改代码中的 `recordings_dir` 变量
